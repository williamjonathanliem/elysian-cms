# VManage CMS

![license](https://img.shields.io/badge/license-MIT-blue.svg)

A customized admin dashboard built with React, Material UI, and Vite.
Designed for efficient room, reservation, and guest management within
the VManage ecosystem.

## Overview

VManage CMS provides a clean and scalable frontend structure for
property and reservation management. The system is optimized for speed,
modularity, and ease of integration with any backend service. This
project is based on the Minimal UI framework and refined to suit the
needs of the VManage platform.

## Available Pages

-   Dashboard
-   Rooms
-   Reservations
-   Guests
-   Settings
-   Authentication (Sign In)
-   Not Found

## Getting Started

### Clone the project

    git clone https://github.com/williamjonathanliem/vmanage.git

### Recommended environment

Node.js v20.x and above

### Install dependencies

    npm install

or

    yarn install

### Run development server

    npm run dev

or

    yarn dev

### Build for production

    npm run build

or

    yarn build

### Local development URL

    http://localhost:5173
